New in version 1.6.0:
Fixed a bug where the exit function would just restart the program!
Restarting the program now clears the console for better usability!
Restart function now asks use if they want to restart the program and exits if they don't!
Added warning popup when program doesn't have admin privileges when started!
New MinecraftServerElevator program builtin to create antivirus exception to improve performance and is run during installation!
Installation folder is now also added to path for power users who would like to run the program from the command line!
